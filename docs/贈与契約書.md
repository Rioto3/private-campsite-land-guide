第1条（贈与の目的）
甲は、上記不動産を何らの対価を受けることなく乙に贈与し、乙はこれを受諾する。

第2条（所有権移転の時期）
上記不動産の所有権は、本契約締結と同時に甲から乙に移転する。

第3条（引渡し）
甲は、令和7年6月30日までに上記不動産を乙に引き渡す。

第4条（登記手続き）
甲は、乙の請求により、乙の費用負担で所有権移転登記手続きに協力する。

第5条（境界の明示）
甲は、乙に対し境界を明示しない。境界に関する一切の調査・確定は乙の責任において行う。

第6条（契約不適合責任）
甲は、上記不動産に関し、契約不適合責任を負わない。

第7条（公租公課の分担）
令和7年度の固定資産税その他の公租公課は、令和7年6月15日をもって甲乙間で日割り計算により精算する。

第8条（費用負担）
本契約に関する登記費用、税金その他一切の費用は乙の負担とする。

第9条（その他）
本契約に定めのない事項については、甲乙協議の上決定する。

本契約の成立を証するため、本書2通を作成し、甲乙それぞれ署名押印の上、各自1通を保有する。

【重要事項】
- 境界非明示条項（第5条）：隣地トラブルのリスクあり
- 契約不適合責任免責（第6条）：土壌汚染等の責任なし
- 費用負担（第8条）：登記・税金すべて受贈者負担
- 実印での押印と印鑑証明書の添付が必要
